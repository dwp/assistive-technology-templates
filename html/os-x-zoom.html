About this document

Before tesing using OS X Zoom, you will need to configure the settings.

  1. Go to System Preferences > Accessibility > Zoom.
  2. Turn on the option for use keyboard shortcuts to zoom. This will enable accurate zooming to the required level for the tests.
  3. Turn on the option for enable hover text.
  4. Select Advanced and then Controls.
  5. Set the Maximum Zoom to 15. This will enable you to know when you are zoomed in to 15x.

This OS X Zoom test is designed to test whether your website or service is likely to work using common functions of zoom software. It will not tell you if your service will work for every user, but it will help you get an understanding as to how well you can expect your service to work for OS X Zoom users.

You should also test your service with real users of OS X Zoom and other assistive technologies.

Text is readable

Confirm that when zoomed in to 15x the text can still be read.

When zooming in, poor fonts or images of text can become unreadable as they become pixelated.

How to check: Text is readable

  1. Use "OPTION + CMD + =" to zoom in to 15x (as per the configuration).
  2. Navigate around the entire page while zoomed in.

What to check: Text is readable

Confirm that:

  • all of the text is still readable

Images are clear enough

Confirm that images which are important for context are clear at 15x.

Important images are those which are required for interactions such as icons, or those which are important for setting context, such as maps.

How to check: Images are clear enough

  1. Use "OPTION + CMD + =" to zoom in to 15x (as per the configuration).
  2. Navigate over each image on the page.

What to check: Images are clear enough

Confirm that:

  • all icons are clear enough to distinguish at 15x.
  • all non-decorative images are clear enough to distinguish at 15x.

Text on hover text

Confirm that all text on the page works using the Hover Text feature.

Hover Text allows the user to view the text under the mouse pointer larger without having to zoom in.

How to check: Text on hover text

  1. Hold down "CMD" and hover over each piece of text on the page. For long paragraphs you can scroll inside the hover text using the trackpad.

What to check: Text on hover text

Confirm that:

  • all text on the page works with the Hover Text feature
  • all text in the Hover Text feature matches the visible text on the page

Images on hover text

Confirm that all important images on the page works using the Hover Text feature.

Hover Text allows the user to view the alt text on images which may be too small for the user to distinguish.

How to check: Images on hover text

  1. Hold down "CMD" and hover over each image on the page. For long alt text you can scroll inside the hover text using the trackpad.

What to check: Images on hover text

Confirm that:

  • all important images either show a text alternative with the Hover Text feature or that a text alternative is available in the visible page content.
  • any alt text which is presented using the Hover Feature is accurate and meaningful
